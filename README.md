# My Mods for Match Tree

## Installation

1. If not already done, install the [Match Tree Mod Loader](https://github.com/BenediktMagnus/MatchTreeModLoader).
1. Download the desired mods from the latest [release](https://github.com/BenediktMagnus/MatchTreeMods/releases/latest).
1. Extract them into the `mods` folder of your Match Tree installation (probably something like `Steam/steamapps/common/Match Tree Demo`).
